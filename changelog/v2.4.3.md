## 中文

### ✨ 新功能

- 精选周刊新增图文精读版：在周刊详情页可进入更适合连续阅读的长文视图，支持章节目录、音频入口和列表浏览切换
- 内容广场支持关键词搜索：文章、播客、视频和推文列表都可以直接输入关键词筛选，搜索条件会同步到链接，方便刷新和分享
- 我的关注文件夹支持重命名：普通文件夹可直接改名，列表、详情和我的源页面会同步更新

### 🔧 体验优化

- 登录状态更持久：网页登录提示与长期会话保持一致，减少仍有有效会话却被提前要求重新登录的情况
- 帮助页反馈入口更准确：FAQ 里的 GitHub 链接改为公开反馈入口，邮箱信息同步更新

### 🐛 修复

- 精选周刊详情页滚动更稳定：向下浏览文章列表时，左侧目录高亮不会再把页面自动带回上方

## English

### ✨ New Features

- Long-form reading for curated newsletters: each issue can now open a focused reading view with chapter navigation, audio access, and an easy switch back to the list view
- Keyword search in Explore: articles, podcasts, videos, and tweets can now be filtered by keyword, with the search saved in the URL for refreshes and sharing
- Rename folders in My Follow: regular folders can be renamed directly, with the updated name reflected across the list, detail panel, and My Sources

### 🔧 Improvements

- Longer-lasting signed-in sessions: web login state now matches the longer session window, reducing cases where users were asked to sign in again too early
- Clearer FAQ feedback channels: the FAQ now points to the public GitHub feedback entry and the updated contact email

### 🐛 Fixes

- More stable newsletter detail scrolling: while reading down the article list, the left table of contents no longer pulls the page back upward
