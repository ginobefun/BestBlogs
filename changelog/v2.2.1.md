## 中文

### ✨ 新功能

**RSS 即添即用**
- 添加 RSS 源不再需要等待人工审核，符合质量要求的源会立即通过
- 通过后几分钟内即可看到首批文章
- 单条添加和 OPML 导入均支持

**内容广场自定义时间范围**
- `/explore` 时间筛选器新增「自定义」选项，可精确指定起止日期

### 🔧 体验优化

**订阅上限大幅提升**
- Pro：500 → **5000**
- Free：50 → **500**
- X 关注同步：1000 → **2000**

### 🐛 修复

- 修复订阅源较多时早报生成偏慢的问题

## English

### ✨ New Features

**Instant RSS sources**
- Add an RSS source and use it right away — qualified sources are approved instantly
- The first batch of articles becomes visible within a few minutes
- Works for both single sources and OPML imports

**Custom date range on Explore**
- `/explore` time filter now offers a "Custom" option for precise start/end dates

### 🔧 Improvements

**Much higher subscription limits**
- Pro: 500 → **5000**
- Free: 50 → **500**
- X synced followings: 1000 → **2000**

### 🐛 Fixes

- Fixed slow daily brief generation for users with many subscriptions
