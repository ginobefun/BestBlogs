## 中文

### ✨ 新功能

**早报设置简化**
- 早报个性化设置页面已精简，保留最常用的配置项
- 新增**早报形式**选项：目前支持"播客版"（音频 + 封面 + 文稿），"简报 + 海报"和"图文卡片"格式即将上线，可提前选择
- 新增**邮件通知**开关，早报邮件推送即将上线

**详情页快速关注**
- 文章、视频/媒体、推文详情页的来源名称旁直接展示"关注 / 已关注"按钮
- 读到喜欢的来源，当页即可关注，无需跳转订阅管理页

### 🛠 管理后台

**用户个性化快照接口**
- 新增 `GET /api/admin/user/personalization-snapshot`，一次性聚合用户的完整个性化数据：兴趣画像、关注来源、订阅 Feed、推荐 Feed、行为日志、阅读历史、书签、划线、日报历史、标签文件夹等
- 供早报生成、运营分析和 AI workflow 使用，替代原有多接口串行调用

**批量资源查询增强**
- 批量获取资源元信息接口单次上限从 50 条提升至 100 条
- 新增 `uiLang` 参数，支持显式指定内容展示语言（覆盖 Accept-Language 头）

## English

### ✨ New Features

**Simplified Daily Brief Settings**
- The Daily Brief personalization page has been streamlined to surface only the most relevant options
- New **Brief Format** option: "Podcast" (audio + cover + transcript) is available now; "Summary + Poster" and "Card" formats are coming soon — select your preference in advance
- New **Email Notification** toggle for upcoming Daily Brief email delivery

**Quick Follow on Detail Pages**
- Article, video/media, and tweet detail pages now show a "Follow / Following" button next to the source name
- Follow a source you like right from the content page — no need to navigate to subscription management

### 🛠 Admin

**User Personalization Snapshot API**
- New `GET /api/admin/user/personalization-snapshot` endpoint aggregates a user's complete personalization data in one call: interest profile, followed sources, subscribed feeds, recommended feeds, behavior logs, reading history, bookmarks, highlights, daily brief history, tag folders, and more
- Designed for ops analysis and AI workflows — replaces multiple serial API calls

**Batch Resource Query Enhancements**
- Batch resource metadata endpoint limit raised from 50 to 100 items per request
- New `uiLang` parameter to explicitly set the content display language, overriding the Accept-Language header
