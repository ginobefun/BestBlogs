## 中文

### 🐛 修复

**早报时区判断错误（影响英文用户与跨时区场景）**
- 修复了英文用户（默认时区 `America/Los_Angeles`）在北京时间上午可能看到「昨日早报」的问题
- 后端早报可访问日期判断统一以 `Asia/Shanghai` 为基准，与早报生成时的时区保持一致
- 前端早报内容项在切换语言后现在能正确重新拉取对应语言资源数据

**精选周刊左侧目录不随内容滚动**
- 移除了破坏 `position:sticky` 的 `ResizablePanelGroup`，改用标准 flex 布局
- 左侧目录现在始终贴合视口顶部，右侧内容滚动时自动高亮对应目录项
- 激活目录项自动滚入可视区，不再需要手动滚动目录
- 修复了暗黑模式下激活态序号背景色对比度不足的问题

**兴趣标签「展开全部」点击无反应**
- 修复了 CSS 折叠方案导致「展开全部」按钮点击后无视觉变化的问题
- 折叠状态下，已选中的第 9+ 个标签会始终保持可见
- 展开按钮现在明确显示「展开更多（还有 N 个）」，一目了然
- 组内标签按话题 → 实体 → 格式排序，话题类优先展示

**Cmd+K 搜索面板按 Esc 无法关闭**
- 修复了打开搜索面板后按 Esc 键没有响应的问题
- Esc 关闭逻辑现在与 Cmd+K 快捷键统一在同一处管理

**移动端 safe-area 遮挡与错位**
- 修复了 iPhone 14 Pro Max 等含 home indicator 设备上底部导航图标和文字被压缩的问题
- 统一了迷你播放器、悬浮工具栏与底部导航的定位计算，所有 safe-area 间距收敛到单一来源
- 文章、播客、视频等详情页叠加播放器和工具栏时，最后一屏内容不再被遮挡

### 🔧 体验优化

- 优化了播客播放器（Sticky Player）播放控制按钮的触控区域，更易点击
- 修复了早报内容项切换语言后资源数据未刷新的问题

## English

### 🐛 Fixes

**Daily Brief timezone mismatch (affects English-locale & cross-timezone users)**
- Fixed an issue where English-locale users (default timezone `America/Los_Angeles`) saw yesterday's brief in the morning (Beijing time)
- Backend brief access date is now anchored to `Asia/Shanghai`, consistent with the brief generation timezone
- Brief content items now correctly re-fetch when the user switches language

**Newsletter sticky TOC not scrolling with content**
- Replaced `ResizablePanelGroup` (which broke `position:sticky`) with standard flex layout
- The left TOC now stays anchored to the viewport and highlights the active section as you scroll
- Active TOC items auto-scroll into view within the TOC panel
- Fixed dark mode contrast issue on active section number backgrounds

**Interest Tag "Expand All" button not responding**
- Fixed a CSS-based collapse approach that caused the expand button to appear non-functional
- Selected tags beyond the first 8 now remain visible while the list is collapsed
- The expand button now shows "Show more (N more)" for clarity
- Tags within a group are sorted by type: topic → entity → format

**Cmd+K search panel not closing on Esc**
- Fixed unresponsive Esc key when the command palette is open
- Esc handler is now co-located with the Cmd+K toggle logic

**Mobile safe-area stacking issues**
- Fixed bottom nav icons and labels being compressed on iPhone 14 Pro Max and similar devices
- Unified safe-area calculations for mini player, floating toolbar, and bottom nav into a single source
- Article, podcast, and video detail pages no longer have content hidden behind overlapping UI layers

### 🔧 Improvements

- Increased touch target size for playback controls (skip back, play/pause, skip forward) in the sticky player
- Brief content items now re-fetch correctly after a language switch
