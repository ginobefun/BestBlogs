## 中文

### 🔧 体验优化

**混合搜索降级增强**

混合搜索（`discover search`）在 Elasticsearch 不可用时，现在会自动降级为 MongoDB 关键词搜索，搜索结果不中断。结果新增站内阅读链接和内容类型标签，展示更完整。

**RSS 解析增强**

修复了部分 RSS 源（如 langchain.dev、venturebeat.com）因携带 BOM 或 DOCTYPE 声明而解析失败的问题，这些源现在可以正常抓取。

### 🐛 修复

**开放能力（CLI）问题修复**

修复了 v2.0.7 发布后测试发现的多项问题：`discover search` 搜索参数映射、`intake` 命令字段兼容性、`auth` 命令状态展示，以及版本号显示异常。

**公共仓库文档更新**

将 Dify Workflow 相关文档和 DSL 文件统一迁入 monorepo 管理，并完成公共仓库的首次全量同步，开发者文档现已覆盖 CLI、Skills、OpenAPI 及 Workflow 等全部开放能力。

## English

### 🔧 Improvements

**Hybrid Search Fallback**

Hybrid search (`discover search`) now automatically falls back to MongoDB keyword search when Elasticsearch is unavailable, keeping search results uninterrupted. Results also include in-app reading links and content type labels for a more complete display.

**RSS Feed Parsing**

Fixed parsing failures for RSS feeds (e.g., langchain.dev, venturebeat.com) that include BOM or DOCTYPE declarations. These sources now fetch correctly.

### 🐛 Fixes

**Open API / CLI Fixes**

Fixed several issues found in post-v2.0.7 testing: search parameter mapping in `discover search`, field compatibility in `intake` commands, status display in `auth` commands, and incorrect version number display.

**Public Repository Documentation**

Migrated Dify Workflow documentation and DSL files into the monorepo, and completed the first full sync to the public repository. Developer docs now cover CLI, Skills, OpenAPI, and Workflows.
