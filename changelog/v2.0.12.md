## 中文

### 🔧 体验优化

**移动端体验改进**

优化了移动端各页面的触控区域与布局，交互更精准、内容显示更完整，修复了 iOS Safari 等浏览器上页面底部内容被工具栏遮挡的问题。早报图文内容展示更干净，消除了部分客户端下的异常文本。

**服务端性能与稳定性**

升级后端缓存架构，兴趣画像、来源列表、内容详情等数据在多服务器节点间保持一致，减少冗余数据库查询，提升整体响应速度。Redis 不可用时服务自动降级，保障可用性。

### 🐛 修复

- 修复语言偏好更改后，推荐内容未即时切换语言的问题

## English

### 🔧 Improvements

**Mobile experience polish**

Improved touch targets and layout across mobile pages for more precise interactions and fuller content visibility. Fixed an issue on iOS Safari and similar browsers where page content was obscured by the browser toolbar. Daily Brief content rendering is cleaner with stray text artifacts removed.

**Backend performance & reliability**

Upgraded the backend cache architecture so interest profiles, source lists, and content details stay consistent across multiple server nodes, reducing redundant database queries and improving response times. The service degrades gracefully when Redis is unavailable.

### 🐛 Fixes

- Fixed language preference changes not being applied immediately to recommendation content
