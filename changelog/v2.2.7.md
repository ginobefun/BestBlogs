## 中文

### 🐛 修复

- 修复了早报生成在 Pro 用户较多时部分用户被误判超时跳过的问题
- 修复了推文 AI 分析结果解析失败（含代码块的推文内容导致 JSON 截断）的问题
- 修复了晚间回顾在当日早报未完成时无法生成的依赖问题

### 🔧 体验优化

- 晚间回顾邮件今日阅读展示上限从 5 篇提升至 10 篇
- 推文现在展示 AI 生成的语义标题，内容更一目了然

## English

### 🐛 Fixes

- Fixed an issue where some Pro users were incorrectly skipped during daily brief generation when many users were processed concurrently
- Fixed a JSON parsing failure that occurred when tweet content contained code blocks, causing AI analysis results to be truncated
- Fixed a dependency issue where the evening review failed to generate if the daily brief had not yet completed

### 🔧 Improvements

- Evening review emails now display up to 10 today's reading articles (previously 5)
- Tweets now show AI-generated semantic titles for easier reading
